# agt

A Gui Toolkit

## Dependencies

- dwhbll (ofc): https://github.com/dawn-winery/dwhbll
- some gl stuff
- libwayland

## Build

```bash
cmake -B build . -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```
Debug so you can report bugs ~~when~~ if you find them
